# Customer Sales Analysis (Week 5 – Pandas Advanced)

## Project Overview
This project focuses on analyzing customer purchasing behavior and sales patterns using advanced data manipulation techniques in Python with Pandas. The objective is to derive actionable business insights through aggregation, filtering, pivot tables, and visualizations.

## Objectives
- Analyze total and regional sales performance
- Identify top customers contributing to revenue
- Understand monthly and seasonal sales trends
- Apply advanced Pandas operations for data summarization
- Create professional visualizations for decision-making

## Dataset
- sales_data.csv – Contains transactional sales data
- customer_churn.csv – Provided for reference 

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis Performed
- Data cleaning and preprocessing
- Date-time feature extraction (year, month, day)
- GroupBy aggregations
- Multi-condition data filtering
- Pivot table analysis
- Sales trend visualization

## Key Insights
- A small percentage of customers contribute to a major portion of total revenue
- Certain regions consistently outperform others in sales
- Sales show clear monthly and seasonal trends
- High-value transactions are concentrated in specific regions
